# 💰 Expense Tracker App

A sleek, command-line based Python application built using Object-Oriented Programming (OOP). This tool allows users to efficiently log, view, track, and delete their daily expenses through an interactive terminal menu.

---

## 🚀 Key Features

* **➕ Add Expense:** Log spending amounts tagged with custom categories.
* **📋 View Expenses:** Display all recorded history in a clean, readable layout.
* **❌ Remove Expense:** Instantly delete any entry by confirming its details.
* **📊 Total Expenses:** Calculate the grand total sum of all spending seamlessly.

---

## 🧠 Core Programming Logic Used

This project showcases fundamental software development concepts:

* **Object-Oriented Programming (OOP):** Utilizes custom Python `Classes`, `Objects`, and the `__init__` constructor method to bundle features safely.
* **Data Structures:** Implements Python `Lists` to stack entries and `Dictionaries` (Key-Value pairs) to structure data.
* **Control Flow Loops:** Uses `while True` for user menu persistence, nested `for` loops for data traversal, and `if-elif-else` logical statements.
* **Built-in Operations:** Features smart typecasting (`int()`), data manipulation methods (`.append()`, `.remove()`), and optimized calculations via `sum()`.

---

## 🛠️ How to Run

1. Make sure you have **Python 3.x** installed.
2. Open your terminal inside the project directory and run:
   ```bash
   python main.py
   ```
